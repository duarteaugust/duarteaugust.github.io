Continue


Este escript foi criado para testar a funçao continue ignorando a ocorrência do número 10 e para tal foi utilizando o seguinte código

script type="text/javascript"
for (var x = 0; x < 20; x++)
{if (x==10){continue}document.write(x+" ");}
/script

For e Break


Este escript foi criado para testar a funçao For e break cancelando a contagem de 0 a 30 exatamente no número 29 e para tal foi utilizando o seguinte código

script type="text/javascript">
for (var x = 0; x < 30; x++)
